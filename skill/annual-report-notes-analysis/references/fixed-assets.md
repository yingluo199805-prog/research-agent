# 固定资产 sheet 规格

原型：比亚迪模型「固定资产」sheet（附注15，5类资产，2018-2025历史+2026-2028预测）。
设 N = 披露的资产类别数（比亚迪为5：永久业权土地/房屋及建筑物/机器设备/运输工具及船舶/办公及其他设备；
A股公司常见4类：房屋及建筑物/机器设备/运输工具/电子及其他设备）。各区块行数随 N 伸缩。

## 表头

```
B1: 附注 {编号} 固定资产 — {N} 类汇总 + 预测勾稽 ({起}-{终})
B2: 单位+颜色规范（见 conventions.md）
B3: 勾稽：NBV = 原价 - 累计折旧 - 减值 ｜ 本期计提 = 平均原价 × 折旧率 ｜ 期末原价 = 期初 + 购置 + CIP转入 - 处置
B5: 科目 | 附注/来源 | 2018A ... 2028E
```

## 区块结构（编号与顺序固定）

**一、期末账面净值（NBV）— N类汇总**
每类一行（历史=年报硬编码；预测=`原价 - 累计折旧 - 减值` 对应行相减）；
合计NBV行（黄底SUM）；"BS披露NBV"行（来源=资产负债表，仅历史）；
校验行（绿底）：`=合计NBV - BS披露`，应=0。

**二、期末账面原价（Gross Cost）**
历史硬编码；预测 `=上年期末 + 当年购置(8.5) + CIP转入(8.4) - 处置(8.6)`；合计行。

**三、累计折旧 期末**
历史硬编码；预测 `=上年期末×(1-处置率8.2) + 本期计提(五)`；合计行。
（处置率近似扣减随资产处置转出的累计折旧——简化假设，备注里说明。）

**四、减值准备 期末**
历史硬编码；预测默认延续期末值或0（按公司历史减值规律定，深红可调）；合计行。

**五、本期计提折旧（P&L）**
历史硬编码（来自附注"本期计提"）；预测 `=(期末原价+期初原价)/2 × 折旧率(8.3)`；合计行。

**六、隐含折旧率反推**
每类一行：`=IFERROR(本期计提/((期末原价+期初原价)/2),"")`，历史公式、预测引用8.3。

**七、资产新旧度**
整体一行：`=累计折旧合计/期末原价合计`，老化趋势指标。

**八、预测假设输入（深红可调）** — 六个子节：
- 8.1 购置增长率：历史列=购置YoY反推公式，预测列=深红硬编码（默认0%或按Capex指引），O列写理由
- 8.2 处置率（处置÷期初原价）：历史反推，预测深红（默认按近3年平均）
- 8.3 折旧率：历史=引用六，预测默认 `=AVERAGE(近2年实际)`；结构变化大的类别（如电池设备占比上升）可深红手调并写理由
- 8.4 在建工程转入预测：**公式联动在建工程sheet 7.2节**（`=在建工程!{cell}`）；若本次不建在建工程表，则降级为深红假设（默认近2年平均）
- 8.5 购置 = 历史硬编码 + 预测 `=上年×(1+8.1)`
- 8.6 处置预测 `=期初原价×8.2`
各子节带合计行。

**备注与说明** — 三段：【勾稽关系】逐条等式；【特殊事项】重分类/集中减值/口径变化（从附注阅读中总结，这是分析师价值所在，不要省）；【预测假设说明】8.1-8.6各节默认值逻辑。

## 易错点

- 附注"本期计提"是含进存货的全口径折旧，与利润表费用不等，校验对象只能是资产负债表NBV；
- 比亚迪"永久业权土地"2022年才从房屋及建筑物拆出——遇到类别口径变化年，在该类别行C列或备注标注；
- 减值出现负值（处置拨回）是真实披露，照抄并备注，不要"修正"。
