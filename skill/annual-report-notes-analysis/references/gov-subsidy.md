# 政府补助分析 sheet 规格

原型：比亚迪模型「政府补助分析」sheet。**本表只做历史，不做预测**（用户明确约定）。
本表的特点是横跨多个附注口径互验：其他收益、营业外收入、递延收益、
非经常性损益、现金流量表，是识别利润含金量的表。

## 表头与布局

与其他表统一：B列起、第5行年份轴（仅实际年份，无E列）。

```
B1: {公司} 政府补助全景分析 ({起}-{终})
B2: 单位：千元（汇总区为亿元） ｜ 数据来源：各年度年报附注 ｜ 制表：广发汽车
```

## 区块结构

**核心指标汇总（亿元）** — 放最上面给人看的总览：
其他收益合计（其中：政府补助 / 增值税加计抵减等）｜
计入当期损益的政府补助（其中：经常性 / 非经常性）｜
递延收益期末余额（当年新增 / 当年转出）｜政府补助现金流入。
全部 `=明细区对应行/100000` 的公式（千元→亿元），不要二次硬编码。

**一、其他收益拆分（千元）**
与日常活动相关的政府补助｜其他（代扣手续费/增值税加计抵减等）｜合计｜
政府补助YoY｜合计YoY｜政府补助占其他收益比重。

**二、计入当期损益的政府补助分类（千元）**
四个去向各一行：与资产相关-计入其他收益｜与资产相关-计入营业外收入｜
与收益相关-计入其他收益｜与收益相关-冲减财务费用（只在披露年份有值）｜
合计｜与资产相关占比｜与收益相关占比。
早年披露不直接拆分的，按其他收益附注推算并加注"根据XX推算"。

**三、递延收益变动（千元）**
滚动表：期初余额｜本年新增｜本年退回｜本年转出(计入当期损益)｜其他变动｜
期末余额(公式)｜期末余额(年报披露值)｜差异校验行(应=0)｜期末YoY｜
当年转出率(转出/期初)。
转出率是预判未来补助释放节奏的核心指标，备注里点评趋势。

**四、非经常性损益中的政府补助（千元）**
计入当期损益的政府补助(非经常性，来自非经常性损益附注)｜经常性(=二.合计-非经常性)｜
合计｜非经常性占比｜经常性占比｜非经常性YoY。
经常性占比上升通常意味着与资产相关补助（产能补贴）分期释放增加——值得在备注解读。

**五、增值税加计抵减等（千元）**
增值税加计抵减｜代扣手续费/其他｜小计；政策依据写成说明行（如"先进制造业5%加计抵减，2023-2027"）。
公司无此项则省略本节。

**六、政府补助现金流口径（千元）**
收到的政府补助现金（现金流量表"收到其他与经营活动有关的现金"附注明细）｜
递延收益新增(参考)｜计入当期损益(参考)｜现金收到/计入损益倍数。
早年未单独披露的留空并注明。

## 易错点

- "计入当期损益的政府补助"≠"其他收益中的政府补助"：前者还含营业外收入和冲减财务费用部分，
  二者差额就是口径陷阱，本表的二、四节互验就是为了抓它；
- 非经常性损益附注的"政府补助"通常已剔除与资产相关分期释放部分，直接对比会差一大块，
  用"经常性=合计-非经常性"倒挤；
- 增值税加计抵减计入其他收益但**不是政府补助**（不进补助附注），汇总区必须分开列示。
