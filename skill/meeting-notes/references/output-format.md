# 纪要输出格式规范

## 一、标准纪要模板

```markdown
# [公司名称] × [调研机构/场合] 调研交流纪要

**时间：** XXXX年X月X日
**主讲人：** XXX（职位）
**主持：** XXX（机构+职位）[可选]
**主题：** [3-5个核心关键词]

---

## 一、核心摘要

▶ **[主题词]：** [核心结论，含关键数据，2-4句话]

▶ **[主题词]：** [核心结论]

（共5-8条）

---

## 二、Q&A 全文梳理

### （一）[业务板块名称]

**Q：[问题精炼表达]**

A：[回答正文]

**[子业务Q&A继续...]**

### （二）[下一业务板块]

...
```

---

## 二、摘要写作规范

### 格式规则
- 以 `▶ **主题词：**` 开头，主题词2-6字
- 主题词后紧跟核心结论，用冒号分隔
- 每条摘要不超过4句话，约80-150字
- 共5-8条，覆盖所有核心板块

### 内容规则
- **结论先行**：第一句直接给出最重要的判断/数字
- **数据支撑**：必须包含至少一个具体数字
- **逻辑完整**：背景→现状→展望三段式
- **不重复**：各条摘要不互相重复核心内容

### 摘要示例（正确）
```
▶ **驱动电机高速增长，2028年目标收入100亿：** 2026年预计收入约20亿+，
长沙工厂已布局13条产线，2027年产值达45～50亿；常州一期产能目标100亿，
预计2026年12月建成；已锁定华为、小米、理想、零跑等头部客户定点订单。
```

### 摘要示例（错误）
```
▶ **驱动电机：** 公司的驱动电机业务发展得很好，未来会继续增长。
（❌ 无数据，无具体判断）
```

## 三、Q&A整理规范

### 问题整理
- 合并同一话题的多个追问为一个完整问题
- 去除"那个"/"就是"等口语填充词
- 保留问题的核心意图，可适当补充背景

### 回答整理
- 去除口语化表达（"就是说"/"那个"/"对吧"）
- 保留所有数字、时间节点、客户名称
- 用**加粗**标注关键数据和重要判断
- 多点回答用①②③编号

### 分组规则
- 按业务板块分组（煤机/汽车/工业智能/资本结构等）
- 每组用二级标题 `### （一）业务板块名称` 标注
- 同一板块内Q&A按逻辑顺序排列（先概述后细节）

## 四、数字与数据格式规范

| 类别 | 格式规范 | 示例 |
|---|---|---|
| 金额（亿元级）| X亿 或 约X亿 | 约33.5亿、约3～5亿 |
| 金额（万元级）| X万 | 约7,000万 |
| 增速 | X% 或 约X% | 约30%、同比增长约10% |
| 市占率 | X% | 约20%市占率 |
| 年份范围 | XXXX～XXXX年 | 2026～2030年 |
| 波动范围 | X～Y | 约40～50亿 |
| 不确定数字 | 约X | 约2亿+、约百亿 |
| 单位 | 中文 | 亿元、万套、万台 |

---

## 五、多份纪要处理规则

当用户提供多份参考文档（"结合这几篇纪要"）：

1. **逐一读取**每份文档，不跳过
2. **建立数据对照表**（内部）：
   ```
   指标       | 原纪要   | 文档1    | 文档2    | 采用值
   SES收入    | 数亿     | 6-7亿   | —        | 6～7亿
   恒达收入   | 约30亿   | 33.5亿  | 约30亿不到| 约33.5亿
   ```
3. **以最新/最详细数据为准**
4. **标注改动来源**（如"依据260116纪要更新"）
5. **集中呈现**：输出修订版纪要，在文后列出"主要改动说明"

## 六、常见格式问题

### 问题1：年份混用
❌ "今年预计收入约35亿，2026年资本开支…"
✅ "2026年预计收入约35亿，2026年资本开支…"

### 问题2：数字格式不统一
❌ "营收7.6亿到7.7亿，接近9.6、9.7的样子"（口语化）
✅ "2025年营收约9.6～9.7亿"

### 问题3：未分组的Q&A
❌ 所有Q&A平铺，无业务板块分组
✅ 按（一）EGR业务 （二）铁芯业务 （三）机器人业务分组

### 问题4：摘要缺乏数据
❌ "驱动电机业务表现不错，未来发展可期"
✅ "2025年预计收入6～7亿，亏损约2亿；2027年实现盈利，规模达50亿"
